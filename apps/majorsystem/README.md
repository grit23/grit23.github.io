
# Major System Trainer (Web App)

A lightweight, offline web app to practice converting words into numbers using the **Major System** mnemonic.

## What's new
- **Reference page (`reference.html`)** with the full mapping and examples.
- **Top-right Reference icon** on the trainer that opens the reference page.
- **Back to app icon** on the reference page.
- **Automatic progress saving** (localStorage). You can jump between pages without losing your score, streak, the current word, or your options.

## How to run
1. Download `index.html`, `styles.css`, `script.js`, and `reference.html` to the same folder.
2. Open `index.html` in your browser.

Your progress persists automatically between page visits.
